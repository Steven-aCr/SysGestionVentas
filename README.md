# Sistema de Gestión de Ventas.

Este proyecto consiste en un sistema de gestión de ventas e inventario diseñado para administrar productos, controlar movimientos de stock y gestionar la información de clientes, proveedores y usuarios del sistema.

El sistema incluye autenticación de usuarios, control de acceso basado en roles y una base de datos que permite registrar movimientos de inventario, documentos de compra y venta, y mantener trazabilidad de las operaciones.

## Objetivo: 
El objetivo del sistema es proporcionar una solución estructurada para el control de ventas e inventario dentro de una empresa, permitiendo registrar entradas y salidas de productos, administrar usuarios del sistema y mantener un historial completo de movimientos de inventario.

## Característcas principales del Sistema:
• Gestión de usuarios y autenticación  
• Control de acceso basado en roles y permisos  
• Administración de clientes y proveedores  
• Registro y control de productos  
• Registro de movimientos de inventario  
• Documentación de compras y ventas  
• Base de datos relacional 

## Áreas del sistema:
1. Gestión de Usuarios y Seguridad
Administra las cuentas del sistema, autenticación, roles y permisos. Permite controlar qué usuarios pueden acceder al sistema y qué acciones pueden realizar dentro de cada módulo.

2. Gestión de Inventario
Permite registrar y administrar productos y categorías. Controla el stock disponible y registra todos los movimientos de inventario como entradas, salidas y ajustes.

3. Gestión de Compras
Gestiona el proceso de adquisición de productos a proveedores. Incluye registro de proveedores, órdenes de compra y recepción de mercancía que posteriormente se refleja en el inventario.

4. Gestión de Ventas
Administra las operaciones de venta del sistema. Permite registrar clientes, generar documentos de venta y descontar automáticamente los productos vendidos del inventario.

## Módulos del sistema

1. **Gestión de Inventario**
   - Administración de productos y categorías.
   - Control de stock disponible.
   - Registro de movimientos de inventario (entradas y salidas).
   - Gestión de descuentos y precios.
   - Alertas de stock mínimo.

2. **Seguridad y Gestión de Usuarios**
   - Autenticación de usuarios.
   - Administración de cuentas del sistema.
   - Gestión de roles y permisos.
   - Control de acceso basado en roles.

3. **Dashboard, Reportes y Notificaciones**
   - Visualización de indicadores del sistema.
   - Generación de reportes de inventario y ventas.
   - Filtrado de reportes por diferentes criterios.
   - Exportación de información.
   - Sistema de notificaciones para eventos importantes.

4. **Procesamiento de Pagos**
   - Gestión de métodos de pago.
   - Registro y validación de transacciones.
   - Confirmación de pagos realizados.
   - Historial de pagos por cliente o pedido.
